Import of VTK models

The ArcPy scripts "Import VTK" and "Import VTK Border" can be used to import triangle meshes encoded in the VTK-ASCII file format into ArcGIS Pro.

"Import VTK" generates ArcGIS Polygons and "Import VTK Border" generates ArcGIS Polylines.

This image shows the result of five imported VTK-files (imported with the tool "Import VTK"). Each file is saved into a separate Polygon-Feature class and represents a stratigraphic surface.

These layers can be used to generate voxels.

This image shows the borders of five imported VTK-files (imported with the tool "Import VTK Border"). Each file is saved into a separate Polyline-Feature class and represents the border of a stratigraphic surface.

These layers can not be used to generate voxels and should only be used to get a first impression of the data.
