Voxel Model Generation

The ArcPy script "Generate Voxel" allows for the creation of voxels from ArcGIS Polygon layers. It accepts an unlimited amount of layers and creates a user-defined amount of voxels between the top and bottom layers of a stratum.

This image shows the naming conventions for the layers. The top-layer of a stratum has to be infront of the bottom-layer, alphabetically. For example, these input layers would create three sets of voxels. One between the layers Layer_1a and Layer_1b. Another one between the layers Layer_1b and Layer_2a. And one between the layers Layer_2a and Layer_2b.

This image shows an example of generated voxels, displayed as points. 15,625 (25 per axis) voxels per stratum have been created.

These voxels do not have the proper symbology yet. To fix this, follow the instructions in the log-file of the tool.

This image shows the voxels with proper symbology. 1,000 (10 per axis) voxels per stratum have been created.

This image also shows voxels with proper symbology. 125,000 (50 per axis) voxels per stratum have been created.

Note: The run-time of this tool is greatly dependent on the number of voxels to be created.
