Installer Creation

To install the GeologicToolbox in ArcGIS Pro, the ArcPy script files have to be moved in a special ArcGIS folder. As a user-fiendly alternative, for Windows systems an installer might be used. The installer is realized using NSIS (Nullsoft Scriptable Install System).

Introduction into NSIS

With the help of NSIS, an executable file (extension .exe) will be generated. Particularly, this file contains all the files to be installed. To generate this executable binary file, an ASCII script file has to be created and to be compiled using the MakeNSISW tool that comes with NSIS.

Syntax of the NSIS script file

The NSIS script file is self-explaining. At the end of the file, the install path inside ArcGIS Pro and a list of files that will be copied to the target folder is given:

SetOutPath $INSTDIR\Resources\ArcToolBox\Toolboxes

File .\GOCAD_import.py
File .\GOCAD_import_color.py
File .\Import_bif2.py
...
