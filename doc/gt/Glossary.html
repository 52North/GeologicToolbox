Glossary of Terms

ArcPy: ArcPy is a Python side package that provides the user with the ability to write scripts, which use the geoprocessing functions within ArcGIS Pro. More info about what ArcPy is.

Borehole: Usually a polyline, consisting of multiple ArcGIS Points.

Feature (ArcGIS): A geological feature such as a Point, Polyline, Polygon or Multipatch. Stored in Feature Classes.

Feature Class: An ArcGIS class that contains features from one and only one geometry (Point, Multipoint, Polyline, Polygon, Multipolygon, Multipatch). More info on Feature Class basics.

Feature Dataset: A set of thematically related feature classes. More info on how to create Datasets.

GOCAD: A modelling software for seismic, geological and reservoir data. More info on the GOCAD software.

Point (ArcGIS): Two dimensional vertex described by data for the x- and y-axis resp. three-dimensional vertex described by data for the x-, y-, and z-axis. More info on the ArcPy Point.

Polygon (ArcGIS): A geometric shape consisting of at least three ArcGIS Points (vertices). With ArcPy, only triangles can be built consistently. More info on the ArcPy Polygon.

Python (in ArcGIS Pro): Python is the programming language of choice for ArcGIS since 9.0. It has been accepted as the scripting language for geoprocessing users. Scripts for ArcGIS Pro are written in Python 3. More info on Python in ArcGIS Pro.

Script (ArcGIS): Besides the ability to write Python code in the "Python" window of ArcGIS Pro, you can also create ArcPy scripts, which can be used to sequence geoprocessing tools into a reusable tool. More info on how to create scripts.

Stratigraphic surface: A surface representing the upper or lower horizon of a stratum (e.g., a layer of soil). Usually made up of ArcGIS Polygons.

Stratum: Represents a volume body of soil. Described by an upper and lower horizon and the type of soil.

Surface: Usually consists of connected triangles. Can also be a Multipatch.

TIN: Triangular Irregular Network. More info on TIN surfaces.

Voxel: A voxel represents a value on a regular three-dimensional grid. Best described as an "3D pixel". Wikipedia definition (voxel)

VTK: A format from the "Visualisation Toolkit" (VTK) software. Aims to be used as an universal format for data interoperability here. More info on the VTK format.

Soil type: Attribute with an ID, a name and preferably a color (e.g., clay, sand, silt, etc.).
